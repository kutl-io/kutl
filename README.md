# kutl

Real-time collaborative document sync engine.

- **kutl-core** — CRDT engine and shared types
- **kutl-relay** — WebSocket sync relay server (AGPL-3.0)
- **kutl-daemon** — File watcher and sync client
- **kutl** — CLI binary
