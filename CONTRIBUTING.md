# Contributing

This repository is a **read-only mirror** of the kutl OSS sources. Active
development happens upstream and lands here in periodic release snapshots,
so the history here is rewritten each release.

- **Found a bug?** Please file an [issue](../../issues).
- **Security report?** See [SECURITY.md](SECURITY.md).
- **Want to contribute code?** Pull requests against this mirror cannot be
  merged — the mirror is overwritten on each release. We're not yet open
  to external code contributions; check back as the project matures.

If you opened a PR before reading this, no worries — close it and file
the change as an issue instead. We'll port substantive fixes upstream
and credit you in the commit when we do.

Thanks for taking a look at kutl.
